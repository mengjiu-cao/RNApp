# 使用了 expo 作为开发框架

This is an [Expo](https://expo.dev) project

## Get started

Install dependencies

   ```bash
   yarn
   ```

### iOS

1. 启动 Metro Bundler
   ```bash
    yarn start
   ```
2. 进入ios目录 运行
   ```bash
    pod install 
   ```
3. 正常编译运行

### android

1. 直接运行该命令
   ```bash
    npx expo run:android
   ```
   或者进入android目录正常编译运行
